# YouQuiz_Project
	
## 프로젝트 설명
YouQuiz 프로젝트는 
대한민국 청소년들의 디지털 문해력 향상을 위한 교육용 웹사이트로써
실제 youtube 영상과 댓글을 보고, 관련된 문제를 풀면서 디지털 문해력을 기르는 서비스이다.

본 레포지토리는 해당 서비스의 동작을 위한 api를 제공하는 backend API 서버로<br/>
서비스 자체 로그인, 회원가입<br/>
학습할 문제의 목록을 확인<br/>
학습할 문제를 확인<br/>
문제를 풀고 정답을 저장<br/>
자신의 정답에 대한 피드백<br/>
등 의 기능을 제공하는 16개의 API를 제공한다.<br/>
